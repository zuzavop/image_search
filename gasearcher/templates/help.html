{% trans "Help" %}

{% blocktrans %}In this study, the goal is to obtain data that should allow observing whether classes associated with images help users with their search. For this reason, it is important to use classes when reformulating the query{% endblocktrans %}.

{% trans "Steps of the study" %}

{% blocktrans %}Your task is to find the image that will be displayed in the lower left corner, using a maximum of two text queries. The classes (phrases displayed under the pictures) in which the individual pictures have been classified should help you in this task.{% endblocktrans %}.

  1. {% trans "Look at the image you are currently looking for (in the lower left corner)" %}.
  2. {% blocktrans %}Formulate a text query (in English) describing the image you are looking for into the text field in the upper left corner{% endblocktrans %}.
  3. {% blocktrans %}Use the Enter keyboard or the Text search button to submit a query{% endblocktrans %}.
  4. {% trans "Look carefully at the search result" %}
    • {% blocktrans %}If you find the image you are currently looking for, send it using ctrl + click on the image.{% endblocktrans %}.
    • {% trans "Otherwise" %}
      1. {% trans "Rephrase the query using using image classes or your own words" %}
        1. {% blocktrans %}Try to find an image that is very similar to the image you are currently looking for (ideally containing the same looking object){% endblocktrans %}.
        2. {% blocktrans %}Choose one of the image classes (phrases below the image) and click on it to add it to the text query{% endblocktrans %}.
      2. {% blocktrans %}Submit the reformulated query as in step{% endblocktrans %} 3.

{% blocktrans %}If you can't find the picture you're currently looking for or don't want to search for it anymore, you can press the Next button and skip the image that was searcher for{% endblocktrans %}.

{% blocktrans %}If you want to enlarge the image and see it in the context of the video from which it is taken, double-click on the given image. However, images are not allowed to be sent when viewed in the context of a video (due to the currently tested method){% endblocktrans %}.

{% blocktrans %}Each image is assigned 10 classes (phrases), which are sorted by relevance (the class that best describes the image is in 1st place) under each image (press the + button to view them all). The colors (from red to green) of the classes correspond to the uniqueness of the class in the dataset. For searching, it is better to use green color classes, as these classes are not very much represented in the dataset and therefore have a greater chance of helping with the search{% endblocktrans %}.
